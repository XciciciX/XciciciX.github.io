Project 1 VR Computer Lab

It is a VR application designed by Unreal Engine 5 in a team of two. It simulates a VR computer lab for Engineering students. In this game, we implement affordance system and use class inheritance and polymorphism principles in designing Blueprints. We used Github for version control and JIRA for manamgement.
